# Open-Source Scope

Lerim's core is Apache-2.0 and stays that way. The local CLI, runtime, self-hosted sync server, native adapters, context DB schema, MCP server, benchmark scripts, and docs remain fully usable without any paid account or hosted service - the local path is never paywalled.

Lerim itself is not a commercial product. Any hosted, managed, enterprise, or model-training offering belongs to Nablo (https://nablo.io), the company behind Lerim; it is a separate business and Lerim does not depend on it.

Contributions to the open-source core are welcome - see the README and CONTRIBUTING guide.
